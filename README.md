# Personal Portfolio Website

A modern and responsive Personal Portfolio Website built using HTML, CSS, and JavaScript.

## Features

- Responsive Design
- Smooth Scrolling Navigation
- About Me Section
- Skills Section
- Projects Showcase
- Resume Download Button
- Contact Information Section
- Hover Effects and Animations
- Mobile-Friendly Layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```text
Portfolio/
│
├── index.html
├── README.md
├── resume.pdf
```

## Projects Included

- Driver Drowsiness Reduction System
- Apartment Navigation System
- E-Notice Board for College Community

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser

## Project Description

This project was developed as part of the CodeAlpha Frontend Development Internship. The portfolio showcases personal information, technical skills, projects, and contact details in a clean and professional layout.
